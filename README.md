# 할 일 관리 웹 애플리케이션

간단한 브라우저 기반의 할 일 관리 애플리케이션입니다.

## 기능

- **오늘의 할 일 섹션**
  - 오늘 날짜에 해당하는 할 일만 상단 카드에 따로 표시됩니다.
- **새 할 일 추가 모달**
  - `+ 새 할 일 추가` 버튼을 누르면 모달이 열리고, 다음 정보를 입력할 수 있습니다.
    - 제목
    - 상세 내용
    - 일자(날짜)
- **전체 할 일 목록**
  - 아래쪽 카드에는 모든 할 일이 날짜 순으로 정렬되어 표시됩니다.
- **로컬 저장**
  - 브라우저 `localStorage` 를 이용해 할 일 목록을 저장하므로, 새로고침해도 데이터가 유지됩니다.

## 사용 방법

1. 이 폴더에서 `index.html` 파일을 브라우저(Chrome 등)로 열어 실행합니다.
2. 상단의 **오늘의 할 일**에서 오늘 날짜의 할 일을 확인할 수 있습니다.
3. 가운데의 **+ 새 할 일 추가** 버튼을 눌러 모달을 연 후, 할 일 정보를 입력하고 **추가하기** 버튼을 누르면 목록에 반영됩니다.
